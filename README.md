# PriceSentinel Bot

Telegram bot that monitors Binance spot prices and sends private, non-spamming alerts and optional morning summaries based on user-defined thresholds and percent moves.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
